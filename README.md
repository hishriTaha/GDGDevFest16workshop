# GDGDevFest16workshop

This is the demo app we created and debugged during the second workshop at GDG DevFest Bizerte 2016 Event -Oct 26-


## How to use it

*You can choose between either clone or download the project source to your PC.*

- To clone the project to your PC, make sure that git is installed on your PC then open your terminal and type
```
git clone https://github.com/hishriTaha/GDGDevFest16workshop.git
```

- Or just click "Download ZIP"

- Open android studio File > open 
Then select the project folder!
